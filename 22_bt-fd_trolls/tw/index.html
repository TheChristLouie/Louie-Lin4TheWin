Tailwind Image Styling

By CoolTeam

To use tailwind:

<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<script src="https://cdn.tailwindcss.com"></script>
</head>

First, around all of the following steps add <div class="square m-5"></div> The square will square the image. m-5 sets the margin giving buffer from the sides of the screen.
To add an image start with the text: <img src="bron.jpg" alt="LeBron is him"
After that add: class="w-1/3 float-left mr-4 mb-4 rounded-lg border-8 border-blue-500"> in order to style whatever image you selected.
The float-left puts the image on the left and can be changed to float-right to put it on the right side. mr and mb set the margin on the right and bottom respectively. border-8 and border-blue-500 puts the light blue border you see around the image.
Then to format the text add: <p class="text-lg" > Completing all of these steps will wrap whatever text is in the <p> </p> brackets around the selected image!
